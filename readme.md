# Employee Management Service

## AWS Amplify
amplify/backend/api/schema.graphql 파일 내  
Table을 작성한 뒤 amplify push 를 통해 업데이트 적용.  
기본적인 CRUD를 Amplify를 통해 간편하게 생성 가능.

- 백엔드 배포 : amplify push
- 웹 호스팅 : amplify publish

## Webpack
HMR 및 번들링을 통한 로딩 속도 단축
